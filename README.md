# 📱 SmartContactManager

A comprehensive, secure web-based application designed to streamline personal contact management with integrated digital payment capabilities.

---

## 📖 Project Overview

**SmartContactManager** is a robust full-stack web application that allows users to organize, manage, and secure their contact information. Built with a focus on usability and security, it provides an intuitive interface for CRUD operations on contacts, while also featuring a unique, integrated donation system via QR code payments.

---

## ✨ Features

* **User Management:** Secure registration and authentication system.
* **Contact CRUD:** Create, view, update, and delete contacts with ease.
* **Smart Search:** Quickly locate contacts with advanced filtering capabilities.
* **Payment Integration:** Built-in donation feature utilizing QR code generation for seamless transactions.
* **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
* **Data Integrity:** Relational data management ensuring secure storage of user and contact information.

The application follows a standard **Model-View-Controller (MVC)** architectural pattern, ensuring a clean separation between the frontend presentation layer and the backend business logic and database persistence layers.

## 🚀 Installation & Setup

### Prerequisites

* JDK 17 or higher
* Node.js & npm
* MySQL Server

### Backend Setup

1. Clone the repository: `git clone [repository-url]`
2. Navigate to the backend folder.
3. Configure your `application.properties` with your MySQL credentials.
4. Run the Spring Boot application: `./mvnw spring-boot:run`

### Frontend Setup

1. Navigate to the frontend folder.
2. Install dependencies: `npm install`
3. Start the development server: `ng serve`

---

## 💡 Usage

1. **Sign Up / Log In:** Create an account to secure your personal dashboard.
2. **Dashboard:** View your existing contacts list.
3. **Manage Contacts:** Use the buttons to add new entries or edit existing ones.
4. **Search:** Use the search bar to filter contacts by name or email.
5. **Support:** Navigate to the 'Donate' section to view the QR code for contributions.
